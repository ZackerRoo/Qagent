# Qagent

Qagent is a research-first stock opportunity agent project for US and China A-share markets.

The current repository contains product and strategy research for building a quant-style agent that can scan markets, generate opportunity cards, define entry and exit plans, monitor watchlists and holdings, and track post-signal outcomes.

## Current Scope

- US stocks and China A-shares
- Daily scanning plus key intraday alerts
- Free or low-cost data sources during development
- Web dashboard plus agent conversation
- Research, alerts, and review workflow before any automated trading
